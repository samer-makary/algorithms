This directory contains test cases with ground-truth for the knapsack problem.

The input files format is
<number-of-items-N> <knapsack-size>
<item-1-value> <item-1-weight>
.
.
.
<item-N-value> <item-N-weight>

The output files format is
<optimal-solution> <integer-you-can-ignore>
A line containing N zeros and ones indicating whether item-i was taken (1) or not (0),
the ordering of the zeros and ones in the line is the same as the ordering of the items in the input file.
